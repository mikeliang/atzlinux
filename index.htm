debian-cn 中文套件

项目介绍

Debian 是一款非常优秀的 Linux 操作系统，但默认安装缺少中国人常用的软件。 本项目的目的是收集这些常用软件，利用自动化脚本在 Debian 上一键安装，节省大家定制 Debian 的时间。
本项目同时承诺 100% 和 Debian 兼容。

目前收录的中文软件包如下：
  • QQ
  • 微信
  • 百度网盘
  • 搜狗输入法
  • 网易云音乐
  • 有道词典
  • WPS
  • 中文字体

安装教程

  1. 下载安装脚本
    2. 打开终端命令行，执行如下命令，下载一键安装脚本。

    wget http://118.24.9.73/debian/download/install-all-single-script.sh

  2. 执行安装
    3. 切换到 root 账号,在终端命令行执行如下命令即可：

    .   ./install-all-single-script.sh

    注：前面是英文的 点 空格 点，表示在当前路径执行该脚本。
  3. 重新登录
    4. 安装完成后，请退出当前登录的图像界面，再重新登录，让所有安装生效。
