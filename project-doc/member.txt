该文档列出参与铜豌豆项目开发的正式成员信息。

示例如下：
字段名称 英文冒号 英文空格 字段值 末尾不留空格
多个成员信息之间留一空行

Name: 中英文、网名、昵称均可
Uid: 参与铜豌豆项目的 id ，一般和 @atzlinux.com 邮箱一致
Email: 多个邮箱地址，用一个英文逗号隔开
GPGid: 使用 gpg --keyid-format 0xshort -K 输出查看 0x 开头的 8 位 gpg key id
       多个 id，用一个英文逗号隔开
Begin：成为项目成员的时间, yyyy-mm-dd
